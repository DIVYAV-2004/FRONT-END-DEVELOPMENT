    • A selector in CSS is a pattern used to target and select specific HTML elements so that styles can be applied to them. In CSS, selectors are generally grouped into five main categories, and within them, there are many types. Here's the breakdown:

      Simple Selectors

      Target elements based on name, class, or ID.

    1. Element Selector → p { } --- targets all p tag elements
    2. Class Selector → .classname{ } --- targets elements with that class
    3. ID Selector → #idName { } --- targets a single element with that ID
    4. Universal Selector → * { } --- targets all elements
    5. Grouping Selector → h1, h2, p { } --- targets multiple selectors together
      Combinator selector

      Select elements based on their relationship with other elements.

    1. Descendant Selector → div p { } --- targets p tag inside div tag
    2. Child Selector → div > p { } --- targets direct p tag children of div tag
    3. Adjacent Sibling Selector → h1 + p { } --- targets first p tag after heading tag h1
    4. General Sibling Selector → h1 ~ p { } --- targets all p tag after h1 tag at the same level
      Pseudo-classe

      Select elements based on a special state.

    1. a:hover → when mouse hovers over a link
    2. input:focus → when input is active
    3. li:first-child → first list item
    4. li:last-child → last list item
    5. :nth-child(2) → second child element
      Pseudo-elements
    1. select and style part of an element.
    2. p::first-line → first line of a paragraph
    3. p::first-letter → first letter of a paragraph
    4. ::before → insert content before an element
    5. ::after → insert content after an element
      Attribute Selectors

      Select elements based on attributes and values.

    1. [type] → targets elements with type attribute
    2. [type="text"] → targets elements with type="text"
    3. [title~="flower"] → attribute contains the word “flower”
    4. [lang|="en"] → attribute starts with “en”
